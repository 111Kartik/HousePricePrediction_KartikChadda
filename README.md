# House Price Prediction - Internship Week 1

This folder contains the complete implementation for the house price prediction project.

## Files
- `analysis.ipynb`: complete notebook with loading, cleaning, models, evaluation, charts, and insights.
- `Housing.csv`: dataset used for the notebook.
- `summary.pdf`: one-page project summary.
- `charts/`: saved chart images.
- `model_metrics.csv`: evaluation metrics.
- `feature_importance.csv`: Random Forest feature importance.

## How to Run
1. Open the folder in VS Code or Jupyter Notebook.
2. Install required libraries: `pip install pandas numpy scikit-learn matplotlib seaborn notebook`.
3. Run `analysis.ipynb` from top to bottom.

Note: The notebook is compatible with the common Kaggle Housing.csv schema: price, area, bedrooms, bathrooms, stories, mainroad, guestroom, basement, hotwaterheating, airconditioning, parking, prefarea, furnishingstatus.
